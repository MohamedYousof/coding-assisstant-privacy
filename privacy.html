Privacy Policy

Welcome to our custom GPT coding assistant. Protecting your privacy is a priority for us. This privacy policy outlines how your data is handled when using this assistant, particularly in connection with GitHub integration.

1. Information We Collect

The GPT coding assistant may collect the following types of information during use:

  • Input Data: Any text or code you provide as input for the assistant is temporarily processed to generate responses. This data is not stored after the session ends.
  • GitHub Data: If connected to GitHub, the assistant may access repository information, file contents, and metadata based on your permissions. No data is accessed without explicit authorization.

2. How We Use Your Information

The information collected is used solely to provide coding assistance. Specifically:

  • To process your requests and generate relevant responses.
  • To interact with GitHub repositories if integrated, based on your instructions.

3. Data Storage and Retention

The assistant does not retain input data or GitHub-related data after processing your request. Any interactions with GitHub repositories are transient and scoped to the duration of your session.

4. Sharing Your Information

Your data is not shared with third parties. Data accessed through GitHub remains confidential and is not transmitted to external services.

5. Security

We implement robust measures to ensure the security of your data during its temporary processing. However, it is your responsibility to safeguard sensitive information when using the assistant.

6. Third-Party Services

GitHub integration is handled through their official API, and all interactions comply with GitHub's terms of service. For more information, please review GitHub's API documentation.

7. Your Rights

You have full control over the input data you provide. You can disconnect GitHub integration at any time and manage permissions through your GitHub account settings.

8. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be communicated to you, and the updated policy will be available here.
